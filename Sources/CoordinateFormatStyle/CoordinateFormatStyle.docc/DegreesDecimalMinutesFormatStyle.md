# ``CoordinateFormatStyle/_LocationEssentials/CLLocationCoordinate2D/FormatStyle/DegreesDecimalMinutes``

Degrees and Decimal Minutes (DDM).

## Overview

The `DegreesDecimalMinutes ` style generates and parses string representations of coordinates using the Degrees and Decimal Minutes format, also known as DDM notation. Use this type to create DDM representations of coordinates and create coordinates from text strings in DDM format.

Degrees and Decimal Minutes (DDM). (DDM) is a notation for expressing latitude and longitude geographic coordinates as decimal fractions of a degree.
